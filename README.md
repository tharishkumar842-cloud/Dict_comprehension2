# Dict Comprehension Practice

This repository contains Python practice files focusing on **dictionary comprehensions**.  
The files included are:

- `dict_comprehension_practice5.py` – Update salaries by 10% if less than 50,000
- `dict_comprehension_practice6.py` – Even numbers are getting transformed into their squares
- `dict_comprehension_practice7.py` –  – Creates a dictionary for numbers 1–10 where each key maps to another dictionary containing:
    - `"square"` → the square of the number
    - `"cube"` → the cube of the number
